# 4. 기능 명세 (User Story)

## 4.1 MVP 범위 (12 영업일, 2인 개발)

### Epic 1: 사용자 인증 (개발자 A — 1일)

- **US-1.1** 이메일 매직링크 회원가입/로그인
- **US-1.2** GitHub OAuth 로그인
- **US-1.3** 가입 시 약관/개인정보처리방침 자동 동의 (페이지 진입 시 동의 간주, 명시 표기)
- **US-1.4** Profile 자동 생성 트리거 (`handle_new_user`)

### Epic 2: 태스크 풀이 (개발자 A — 5일)

- **US-2.1** 태스크 목록을 카테고리/난이도로 필터링
- **US-2.2** 태스크 상세에서 요구사항/제약 확인
- **US-2.3** AI(Claude Sonnet 4.6)와 대화하며 결과물 작성
- **US-2.4** AI 응답의 코드 블록을 결과물에 추가
- **US-2.5** 결과물 직접 편집 가능 (단일 버전, MVP)
- **US-2.6** 시도 횟수/시간/토큰 사용량 실시간 확인
- **US-2.7** 최종 결과물 제출 및 채점 트리거

### Epic 3: 평가 & 피드백 (개발자 B — 5일)

- **US-3.1** 단계별 채점 진행 상황 실시간 확인 (Realtime)
- **US-3.2** 100점 만점 점수 + 항목별 점수 + 백분위 확인
- **US-3.3** 잘한 점/개선할 점 피드백 + 신뢰도 메타 노출
- **US-3.4** 풀이 히스토리 조회
- **US-3.5** 채점 이의 신청 가능

### Epic 4: 운영/안정성 (공유 — 1일)

- **US-4.1** Rate limiting (사용자당 일일 한도) — B
- **US-4.2** 에러 처리 + 사용자 친화적 메시지 — A
- **US-4.3** 이용약관 / 개인정보처리방침 페이지 — B
- **US-4.4** Sentry/로깅 — B
