# 1. 서비스 개요 & 목표

## 1.1 서비스 정의

**PromptQuest** — 개발자가 실제 업무 시나리오 태스크를 Claude와 함께 풀고, AI 활용 능력을 정량적으로 측정하고 개선할 수 있는 웹 플랫폼.

> "AI를 잘 쓰는 개발자가 점수로 증명되는 곳"

각 태스크는 하나의 *quest*로, 사용자가 프롬프트로 AI를 다뤄가며 풀이하고 점수를 받습니다.

## 1.2 차별화 포인트

- **객관적 점수화** — 4단계 평가 파이프라인으로 신뢰성 있는 채점
- **Cross-family 평가** — 사용자는 Claude, 평가는 OpenAI로 자기 채점 편향 회피
- **실전형 태스크** — 실제 개발 업무 시나리오 기반 문제

## 1.3 타겟 사용자

- 1차: AI 도구를 업무에 활용하는 주니어/미들 개발자
- 2차: AI 활용 교육이 필요한 팀/회사 (B2B 확장)

## 1.4 성공 지표 (MVP)

| 지표              | 목표                             |
| ----------------- | -------------------------------- |
| 가입자 수         | 출시 후 1개월 100명              |
| 태스크 완료율     | 가입자 중 50% 이상 1개 이상 완료 |
| 재방문율          | 1주 내 재방문 30%                |
| Judge AI 일관성   | `inter_run_stddev < 5`           |
| 골든 셋 평균 오차 | `golden_set_mae < 8`             |
