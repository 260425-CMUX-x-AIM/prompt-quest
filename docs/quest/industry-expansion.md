# 업종 확장 Quest 22종

현재 `/tasks`에 있는 문제들은 전반적으로 웹서비스, SaaS, 협업툴, 커머스 문맥에 강합니다. 그 방향은 유지하되, 실제로 더 넓은 사용자군을 잡으려면 산업별 실무 맥락이 더 분명한 문제군이 필요합니다.

이 문서는 PromptQuest의 기존 평가 철학을 유지하면서도 의료, 보험, 물류, 제조, 교육, 공공, 부동산, 호텔, 법무, 에너지 등 더 다양한 업종으로 확장할 수 있는 quest 후보를 정리한 문서입니다.

## 확장 원칙

- 업종 맥락이 문제 설명만으로 바로 느껴져야 함
- 실제 현업에서 AI에게 맡길 만한 산출물이어야 함
- 단순 지식 퀴즈가 아니라 정리, 분류, 진단, 검토, 복구 지시 능력이 드러나야 함
- 공개 테스트와 숨김 체크를 설계할 수 있어야 함
- 기존 카테고리 `analysis / communication / workflow / implementation / diagnosis / review / security / strategy`에 자연스럽게 들어가야 함

## 현재 비어 있는 산업군
