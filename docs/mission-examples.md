# 미션 문제 예시 16종

태스크 카테고리별로 활용 가능한 실전 시나리오 예시. MVP에서는 일부만 시드로 사용하고, 나머지는 v1.5 이후 콘텐츠 확장 시 참고.

## 초급 (Easy)

### 1. 정규식 만들기

- **입력:** 텍스트
- **과제:** 텍스트에서 특정 패턴(이메일/전화번호 등)을 추출하는 정규식 작성
- **평가:** 정확한 매칭, 엣지 케이스 처리, ReDoS 방어

### 2. 짧은 회의록 요약하기

- **입력:** 텍스트 (회의 대화문)
- **과제:** 10분짜리 회의 대화문을 읽고 3줄 요약 + 액션 아이템 3개로 정리
- **평가:** 핵심 파악, 불필요한 내용 제거, 출력 형식 준수

### 3. 장황한 이메일을 깔끔하게 바꾸기

- **입력:** 텍스트
- **과제:** 감정적이고 길게 쓴 이메일을 공손한 비즈니스 메일로 재작성
- **평가:** 톤 조절, 정보 보존, 목적 명확화

### 4. 문서에서 정보 추출하기

- **입력:** 텍스트 또는 PDF
- **과제:** 긴 공지문에서 날짜 / 장소 / 비용 / 준비물만 표로 추출
- **평가:** 정확도, 구조화 능력, 누락 여부

### 5. 초보자용 설명 만들기

- **입력:** 텍스트
- **과제:** 어려운 기술 설명을 중학생도 이해할 수 있게 풀어서 설명
- **평가:** 난이도 조절, 비유 사용, 정확성 유지

### 6. SNS 카피 3종 만들기

- **입력:** 제품 설명 텍스트
- **과제:** 같은 내용을 인스타용 / 링크드인용 / X용으로 각각 다르게 작성
- **평가:** 채널별 톤 변환, 간결성, 설득력

## 중급 (Medium)

### 7. 회의 음성 듣고 인사이트 내기

- **입력:** 오디오 파일
- **과제:** 회의 내용을 듣고 요약 + 주요 쟁점 + 다음 액션 + 숨은 리스크 정리
- **평가:** 단순 요약을 넘는 해석 능력, 실행 가능성

### 8. 여러 문서 비교 후 추천안 만들기

- **입력:** PDF 2~3개
- **과제:** 제안서/정책문서 여러 개를 비교하고 추천안 1개 제시
- **평가:** 비교 기준 설정, 근거 제시, 의사결정 논리

### 9. 고객 인터뷰에서 페인포인트 뽑기

- **입력:** 인터뷰 대화문 또는 음성
- **과제:** 인터뷰 내용에서 사용자 문제, 반복되는 불만, 개선 아이디어 추출
- **평가:** 패턴 인식, 정성 데이터 해석, 인사이트 품질

### 10. 엉망인 데이터에서 보고서 만들기

- **입력:** CSV/엑셀
- **과제:** 판매 데이터나 설문 데이터를 보고 핵심 지표 + 이상치 + 시사점 작성
- **평가:** 수치 해석, 우선순위, 보고서 구조

### 11. 원하는 이미지를 정확히 프롬프트로 만들기

- **입력:** 텍스트 브리프
- **과제:** "친환경 프리미엄 음료 광고 이미지" 같은 요구를 이미지 생성용 프롬프트로 작성
- **평가:** 시각적 구체성, 스타일 통제, 불필요한 모호성 제거

### 12. 사진 보고 상품 상세페이지 문구 만들기

- **입력:** 이미지 1~3장
- **과제:** 제품 사진만 보고 상품명 / 핵심 특징 / 상세 설명 / 구매 포인트 작성
- **평가:** 관찰력, 상업적 문구 감각, 과장 최소화

## 고급 (Hard)

### 13. 모순된 요구사항을 정리해 실행 프롬프트 만들기

- **입력:** 텍스트 브리프
- **과제:** 서로 충돌하는 요구사항이 섞인 요청을 읽고 모순 정리 + 가정 명시 + 실행용 프롬프트 작성
- **평가:** 요구사항 정제, 리스크 관리, 프롬프트 설계력

### 14. 사고 대응 포스트모템 작성하기

- **입력:** 로그, 채팅, 장애 타임라인
- **과제:** 자료를 바탕으로 문제 원인 / 영향 / 재발 방지책 문서 작성
- **평가:** 근거 기반 추론, 책임 회피 없는 서술, 구조화

### 15. 하나의 입력으로 여러 산출물 만들기

- **입력:** 회의 음성 + 관련 문서
- **과제:** 같은 자료로 경영진 요약본 / 실무자 액션리스트 / 고객 공지 초안 3개 생성
- **평가:** 독자별 톤 조절, 목적 맞춤형 출력, 멀티아웃풋 설계

### 16. 멀티모달 에이전트 프롬프트 설계하기

- **입력:** 텍스트 + 파일 + 이미지
- **과제:** "문서 읽고, 표 추출하고, 요약하고, 슬라이드 초안까지 만드는" 작업을 위한 시스템 프롬프트/워크플로 프롬프트 작성
- **평가:** 단계 분해, 도구 사용 지시, 예외 처리, 출력 계약 명시
